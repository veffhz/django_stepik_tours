travel demo project with Django
==================

##### features:
 - index page
 - direction page
 - tour page
 
##### requirements:
 - Python 3.6+
 - Django 3

##### install requirements:
`pip3 install -r requirements.txt`

##### run app:
 - run `./manage.py runserver`
 - open default page http://127.0.0.1:8000/
